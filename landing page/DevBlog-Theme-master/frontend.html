Front end

  • App is a stateful React component

    • Data state variable set to an empty array

  • CallApi class extends EventEmitter

    • Separates asynchronous calls from synchronous App class

    • Fetches the data from the GET endpoint and sets it equal to the Data state variable in App

  • handleSubmit makes asynchronous call to the POST endpoint, set the state equal to the users input

  • Data passed to Table and Graph components as props

    • Table made from React-Table library

      • Memorizes the data

      • Column headers and rows created using map function

    • Graph made from React-Vis library

      • Bar chart: report date on x-axis, confirmed cases on y-axis

  • Modals provide more info about app

    • Show state is toggled when button is clicked
